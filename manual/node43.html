Using other path names for the data sets.

All pathnames to data locations are assigned in igmt_configure.tcl. All raster data sets have two variables assigned to them: raster_data(i) and raster_colormap(i), they refer to the location of the raster data set number i and the location of the default colormap, respectively. Simply add a line like set raster_data(3) "/home/user/gtopo30/topo_6.2.img" to your igmt_siteconfig.tcl file, if your GTOPO30 dataset (number 3 internally) is in /home/user/gtopo30/. The equivalent variables for polygon data are called poly_data(i), and it is simplest to search for the appropriated variable names in igmt_configure.tcl.

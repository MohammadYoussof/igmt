Item Projection

The projection order chosen for the dialog box follows the GMT manual
URL: http://www.soest.hawaii.edu/wessel/gmt/gmt_doc.html

closely. Projections themselves are explained briefly in the pscoast man page. The last check-box, ``custom projection'', allows the user to specify the projection with the magnification factor in the GMT format explicitly. This might be needed since formatting is not perfectly done by iGMT and not all GMT projections are implemented. Some of the projections adjust the geographic region to be plotted as suitable.
